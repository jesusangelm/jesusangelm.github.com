Si eres de los que estas aprendiendo la nueva version 5 del lenguaje HTML que esta por salir dentro de unos años, aprendiendo CSS3 o simplemente andas buscando un buen editor web del tipo WYSIWYG entonces tal vez te interese esto.

BlueGriffon es un nuevo editor web del tipo WYSIWYG, es decir "lo que ves es lo que obtienes" que utiliza el motor Gecko (Motor de renderizado de Firefox 4). Este editor tiene entre sus caracteristicas el soporte para la nueva version del lenguaje web HTML conocida como HTML5 y tambien cuenta con soporte para CSS3 asi como un editor SVG lo que sin duda te podria ayudar a aprender el nuevo lenguaje HTML5 y estilizar tus web con CSS3 :)

Para instalar BlueGriffon en Ubuntu tan solo debemos descargarnos el instalador, lo que podriamos hacer desde la terminal tecleando:
<pre lang="bash" line="1" escaped="true">wget http://bluegriffon.org/freshmeat/nightlies/latest/BlueGriffon-0.8.1-Linux-x86_64-Install</pre>
Una vez descargado el archivo <strong>BlueGriffon-0.8.1-Linux-x86_64-Install </strong>le damos click izquierdo y elegimos <strong>Propiedades</strong>, luego elegimos la pestaña "<strong>Permisos</strong>" y click en donde dice "<strong>Permitir ejecutar archivos como un programa</strong>", por ultimo le damos doble click al archivo e instalamos desde el asistente de instalacion que se nos mostrara.

Ya con esto tendras BlueGriffon instalado y podras comenzar a jugar con el y realizar pagians en HTML5 y CSS3 :)
